# Qalm — Hospital Queue Management

Qalm eliminates waiting room chaos for private clinics. Patients scan a QR code, get a token number, and track their queue position in real time on their phone. No app download needed.

## Live App
https://hospital-queue-kappa.vercel.app

## Features
- Real-time queue tracking
- Multi-department support
- Doctor dashboard
- WhatsApp reminders
- Multi-language: English, Hindi, Kannada, Tamil
- Analytics dashboard
- Patient visit history

## Tech Stack
React, Firebase (Firestore + Auth), Framer Motion, Vercel
